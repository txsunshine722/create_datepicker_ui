# Datepicker UI (Static)

This project is a static Datepicker UI built with only HTML and CSS.

It was created as a frontend practice exercise to improve layout, positioning, and styling skills without adding JavaScript functionality.

## Project Page

https://roadmap.sh/projects/datepicker-ui

## What The Project Is About

- Creating a clean, reusable datepicker interface mockup.
- Practicing CSS layout techniques such as Flexbox and Grid.
- Using absolute positioning for a floating datepicker panel.
- Styling interactive-looking UI elements like date buttons, month navigation arrows, and tags.

## How It Was Built

- Structured the UI in `index.html` with:
	- A date input field and icon button (trigger row)
	- A month selector with left/right chevrons
	- A calendar grid with days and date cells
- Styled everything in `css/style.css` using:
	- CSS custom properties (variables) for color theme
	- Flexbox for horizontal groups and alignment
	- CSS Grid for the 7-column calendar layout
	- Absolute positioning to place the calendar panel below the input trigger
- Imported Google Fonts and Material Symbols for typography and icons.

## Notes

- This is intentionally a non-functional component (UI-only).
- JavaScript can be added later to make date selection and month navigation work.
